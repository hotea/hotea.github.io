使用 github 和 hexo 搭建 blog

  1. 安装 hexo, nodejs, git
    npm -g install hexo-cli

  2. 初始化博客主目录
    hexo init Myblog
    cd Myblog
    npm install

  3. 配置
    _config.yml

  4. 生成, 部署
    hexo g 生成
    hexo s 本地测试
    hexo d 部署
